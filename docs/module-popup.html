Module: popup

Data

Name Type Default value Description
newPreset String "" Stores input value for creating preset
preset Preset - Object that holds a name(Strings.name), openlist input(Strings.openInput), blacklist input(Strings.blockInput), color of its button(color), boolean for its on/off state(value), list of websites to open(openlist), and list of tabs to block(blacklist)
website Website - Object that holds a website url(site) and a boolean for on/off toggle(enabled)
list Array - An array which is the highest level data storage within the vue. Holds all the presets
index Int 0 Tracks the currently active preset within the list
appOn Bool false Holds the on/off state of the extension as a boolean

Events

Name Payload Type Description
togglePreset Toggles a preset on/off
setPreset Creates a new preset from the newPreset input string, or turns that preset on if it exists already
togglePreset Toggles a preset on/off
removePreset Removes a preset from the list
addOpenlistSite Adds a website from the openInput string to a preset's openlist
addBlacklistSite Adds a website from the blockInput string to a preset's openlist
removeSite Removes a website from a preset's openlist/blacklist
getData Updates current fields with those in chrome storage
trackChange Delay-based recursive function to deal with asynchronous storage
storeLocalList updates chrome storage with current list
storeLocalIndex updates chrome storage with current index
storeLocalEnabled updates chrome storage with current appOn value
appEnable toggles appOn, turns the app on or off
refresh refreshes data, called after data is stored
encode encodes preset names for html id parsing, allows for any ascii preset name
