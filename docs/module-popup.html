Data

Name Type Default value Description
newPreset String "" Stores input value for creating preset.
preset Preset - Object that holds a name(Strings.name), openlist input(Strings.openInput), blacklist input(Strings.blockInput), color of its button(color), boolean for its on/off state(value), list of websites to open(openlist), and list of tabs to block(blacklist).
website Website - Object that holds a website url(site) and a boolean for on/off toggle(enabled).
list Array [] An array which is the highest level data storage within the vue. Holds all the presets.
index Int 0 Tracks the currently active preset within the list.
appOn Bool false Holds the on/off state of the extension as a boolean.
calMode Bool false Indicates if the calendar shall be shown or hidden.
name String "" Input from the modal that renames a preset.
nameState Bool null State to indicate if user input into rename modal is valid.
curRow String "" # concatenated with the HTML ID of the preset/row from which the user invoked a request to delete a preset.
curPreset Int -1 Key of the preset/row from which the user invoked a request to delete or rename a preset.
componentKey Int 0 Key used to force update the calendar component when changes are made to presets.
selName String "Select a mode" Default text displayed in calendar component's Mode selection dropdown.
newPreset string "" Synced to the input form for adding a preset.

getData None Updates current fields with those in chrome storage.
