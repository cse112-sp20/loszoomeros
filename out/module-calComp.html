Module: calComp

Data

Name Type Default value Description
eventData ScheduledEvent - Object simulating a scheduled preset event. Holds a startTime, the time a preset will turn on; endTime, the time a preset is scheduled to turn off (not yet implemented); calDate, the calendar date for when a preset is supposed to turn on; preset, the preset that is being scheduled to turn on.
list Array - An array which is the highest level data storage within the vue. Holds all the presets.
scheduleList Array - An array for storing the eventData objects.
recur Bool false Holds the on/off state of the extension as a boolean

Events

Name Payload Type Description
storeSched Stores the vue input fields as a scheduled event. Filters out duplicated start times/dates.
clearSched Clears all scheduled alarms
updateSel Used to grab the value from the select and store it in eventData
toggleRecur Function passed into the recurrence toggle
getData Updates current fields with those in chrome storage
trackChange Delay-based recursive function to deal with asynchronous storage
updateSelect Function used to generate the preset selector dropdown
test Function used to generate the preset selector dropdown
